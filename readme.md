## ChatApp Backend

### Technologies
 - NodeJS
 - JavaScript
 - Express
 - Bcrypt
 - JsonWebToken
 - Mongoose
 - socket.io

 ### Apis

  - POST - /auth/register
      - username (unique)
      - password
      
  - POST - /auth/login
      - username
      - password

  - GET - /messages
      - sender
      - receiver

  - GET - /users
      - currentUser
